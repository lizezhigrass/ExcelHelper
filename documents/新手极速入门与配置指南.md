# 🚀 Excel 快捷查询助手 — 新手极速入门与配置指南

欢迎使用 **Excel 快捷查询助手**！这是一个专门为了解放您的双手而设计的 Excel 辅助小工具。
如果您每天需要把 Excel 表格里的数据（如“物资名称”、“零件代码”）拿去数据库或其他表格里查出“库存、规格、单价”，然后再一个个复制粘贴回 Excel，那么这个软件就是您的救星！

## 💡 一分钟看懂：这软件有什么用？

本软件采用**“常驻托盘 + 全局快捷键”**的设计，它能够在不打扰您正常工作的前提下，提供以下三大核心神器功能：

| 序号 | 核心神器 | 适用场景 | 新手一句话解释 |
| :--- | :--- | :--- | :--- |
| **1** | **极速弹窗回写** | 快速录入单行数据 | 选中 Excel 某一行，按下快捷键，自动根据当前行的内容去后台模糊查询，弹窗展示结果。**双击某行，数据就自动填入 Excel 对应格子，窗口自动关闭**！ |
| **2** | **自定义高级面板** | 复杂筛选与精准控制 | 觉得自动查询不够灵活？按下快捷键弹出一个包含文本框、数字范围、下拉菜单的高级面板。一边在 Excel 中移动光标，一边在面板中双击回写！ |
| **3** | **批量自动填充** | 数百上千行批量作业 | 选中 Excel 中很多行，按一下快捷键，软件会在后台**自动对每一行进行查询**，并将第一条匹配结果**秒级批量填满整个 Excel 表格**！ |

## 🛠️ 新手第一步：如何安装并跑起来？

由于本软件使用 Python PyQt5 编写并利用 `xlwings` 与 Excel 实时交互，您的电脑只需要满足以下极简条件：

1. **操作系统**：Windows 操作系统。
2. **Excel 软件**：建议安装并打开 Microsoft Excel 或 WPS Office（确保表格处于开启状态）。

### 极速运行步骤：
1. **下载或解压项目文件**到电脑的某个文件夹下（例如 `D:\Excel_Helper`）。
2. 双击运行文件夹中的 `start.bat` 文件。
3. 您会看到屏幕右下角的系统托盘（任务栏通知区域）出现了一个精美的**表格小图标**，同时右下角弹出气泡通知，提示您软件已成功启动并就绪！

## ⚙️ 核心概念：如何配置成您自己的工具？

想要查您自己公司的数据库或表格？没问题！
用鼠标**右键点击**右下角托盘图标，选择 **「设置」**（或双击托盘图标），即可打开全可视化的设置中心。

本软件的配置体系非常像“连线游戏”，主要由以下 4 个核心要素组成，我们通过一个生动的图解来认识它们：

```mermaid
graph TD
    A[Excel表格活动行] -->|1. 根据映射读入| B(输入映射 Input Mappings)
    B -->|2. 送入配置好的数据源| C[查询模块 Query Modules]
    C -->|3. 得到查询结果后| D(输出映射 Output Mappings)
    D -->|4. 双击自动写回| E[Excel表格对应列]
```

### 1. 查询模块 (Query Modules) — 数据源头
> 告诉软件：**“我去哪里查数据？”**

在设置中心的「查询模块」选项卡中，您可以添加并管理数据源。目前支持两种新手最爱的数据源：
* **Excel 文件数据源 (`excel_file`)**：只需选择您电脑上的某个 `.xlsx` 备件库或产品名录文件，指定读取哪一个工作表（如 `Sheet1`）。
* **SQLite 数据库源 (`sql_table`)**：只需选择一个本地的 `.db` 数据库文件，指定查询哪一张表。

### 2. 交互模块 (Interaction Modules) — 用户界面
> 告诉软件：**“我的查询窗口长什么样？”**

您可以为不同的业务定制不同的界面类型：
* **`search_panel`**：包含几个简易文本框的搜索面板。
* **`display_panel`**：最高级的自定义面板，可以配置文本框、下拉菜单（如“单位”）、数字范围框（如“库存数量范围”）、日期筛选器等。
* **`auto_fill`**：无需界面的批量自动填充模块。

### 3. 映射组 (Mapping Groups) — 核心纽带
> 告诉软件：**“输入和输出的列如何与数据库格子对齐？”**

这是最关键的一步，就像把 Excel 的列 and 数据库的字段用线连起来：
* **输入映射 (Input Mappings)**：告诉软件，从 Excel 的哪一列（比如 `A` 列物料代码）读取数据，输入给查询界面的哪一个搜索框。
* **输出映射 (Output Mappings)**：告诉软件，查到结果后，把数据的哪一项（比如 `price`）自动写入 Excel 的哪一列（比如 `F` 列单价）。
* **展示列 (Display Columns)**：决定弹窗里显示哪些列，列宽是多少。

### 4. 快捷键设置 (Hotkeys) — 一键唤醒
> 告诉软件：**“按什么键盘键能触发这个映射组？”**

在「快捷键设置」中，您可以给不同的映射组绑定一个全局快捷键：
* 比如，给「备件查询映射组」绑定 `Alt + X`。
* 比如，给「批量填充映射组」绑定 `Alt + A`。
* 配置完成后，点击「保存并关闭」，软件即可在后台静默监听该快捷键。

## 📝 新手常见问题与排错手册

> [!WARNING]
> **Q1：按下快捷键没有任何反应，右下角提示“未检测到打开的 Excel 工作表”？**
> * **解答**：这通常是因为您没有打开任何 Excel 表格，或者您的 Excel 表格处于“正在编辑公式/正在输入文字”的挂起状态。请确保先用鼠标单击一下 Excel 表格的某个空白格子，再按下快捷键。

> [!NOTE]
> **Q2：双击表格行后，数据没有填回 Excel？**
> * **解答**：请检查您的「映射组设置」中的 **「输出映射」** 是否配置正确。必须指定 `结果字段`（如 `单价`）对应写入 Excel 的哪一个 `列字母`（如 `F`）。如果没有配置输出映射，双击将仅供查看，不会写入 Excel。

> [!TIP]
> **Q3：为什么我新增了一个映射组，在快捷键下拉菜单里找不到它？**
> * **解答**：我们已经彻底解决了该体验痛点！现在，当您在「映射组设置」中添加并保存好映射组后，**只需切换回「快捷键设置」标签页，下拉菜单就会自动刷新并完美同步**，您无需重新打开设置窗口！
